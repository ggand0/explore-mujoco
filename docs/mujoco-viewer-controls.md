# MuJoCo Viewer Controls

## Mouse

| Action | Control |
|--------|---------|
| Rotate view | Left-click + drag |
| Pan view | Right-click + drag |
| Zoom | Scroll wheel |
| Move in depth | Ctrl + right-click + drag |
| Apply force to body | Double-click on body + drag |

## Keyboard

### Simulation

| Key | Action |
|-----|--------|
| Space | Pause/unpause |
| Backspace | Reset to initial state |
| Enter | Step forward (when paused) |
| `[` | Decrease simulation speed |
| `]` | Increase simulation speed |
| ESC | Quit |

### UI Panels

| Key | Panel |
|-----|-------|
| F1 | Help |
| F2 | Simulation stats |
| F3 | Profiler |
| F4 | Sensors |
| F5 | Visualization options |
| Tab | Toggle visualization modes |
